@TheTimesofMalta Twitter

The Times of Malta maintains a Twitter handle named @TheTimesofMalta which it uses to tweet some of the news items on its web site.

In the past there used to be an RSS feed which has not been turned off. This feed aims to fill that gap by converting the tweet stream into an RSS feed. All images, text and links are the ones found in the original web site.
